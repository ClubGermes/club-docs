# Обзор Club Manager

**Club Manager** — это веб-приложение (SPA) для администраторов и менеджеров клуба.

## Технологический стек

*   **UI Framework:** Svelte 4
*   **Build Tool:** Vite 5
*   **Styling:** Tailwind CSS + DaisyUI
*   **State Management:** Svelte Stores (Native)

## Особенности архитектуры

### Режим работы
Приложение работает как классическое SPA (Single Page Application). Хотя в репозитории присутствуют следы **Tauri** (папка `src-tauri`), в текущем процессе работы они **не используются**. Сборка и деплой осуществляются как для обычного веб-сайта.

### Legacy HTTP Client
В проекте используется собственная обертка над `XMLHttpRequest` (`src/helpers/request.ts`).

> **Важно:** Токен авторизации передается **в теле запроса** (поле `_token`), а не в заголовке `Authorization`. Это критически важная деталь для отладки сетевых запросов.

### Bindings (Связывания)
Приложение использует кастомные заголовки ответа (`x-binding-messages`, `x-binding-notifications`) для автоматического обновления счетчиков непрочитанных сообщений и уведомлений при каждом ответе сервера.
