# Club Manager: Обзор и Стек

**Club Manager** — это административная панель для управления Клубом (клиентами, событиями, встречами). Приложение построено как гибридное решение: оно может работать как в браузере, так и в виде нативного десктопного приложения.

## Технологический стек

Проект использует современный стек на базе Svelte 4 и TypeScript, упакованный в Tauri.

| Компонент | Технология | Версия | Описание |
|---|---|---|---|
| **Core** | **Svelte** | 4.x | Основной UI-фреймворк. |
| **Build Tool** | **Vite** | 5.x | Сборщик и Dev-сервер. |
| **Language** | **TypeScript** | 5.x | Строгая типизация. |
| **Mobile Runtime** | **Capacitor** | 7.x | (Опционально) Возможность сборки под Android/iOS. |
| **Styling** | **TailwindCSS** | 3.x | Утилитарные классы. |
| **UI Kit** | **DaisyUI** | 4.x | Компоненты поверх Tailwind (тема `emerald`). |

## Основные библиотеки

В проекте используется ряд специфических решений вместо стандартных библиотек:

*   **Routing:** Собственная реализация (`src/libs/Router`). Не используется `svelte-routing` или `svelte-kit`. Поддерживает Layouts и Guards.
*   **State Management:** Нативные **Svelte Stores** (`writable`, `readable`, `derived`).
*   **Charts:** `chart.js` + `chartjs-plugin-datalabels` для аналитики и воронок продаж.
*   **Date Picker:** `date-picker-svelte`.
