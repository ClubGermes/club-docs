# Ключевые компоненты (Club Manager)

Приложение `club-manager` — это инструмент CRM и аналитики.

## 1. Dashboard / Monitor (`views/Home.svelte`)

Самый сложный компонент системы.
*   **Функционал:** Воронка продаж, статистика по пользователям ("У агента", "Соискатель", "Член клуба").
*   **Визуализация:** `Chart.js` (`src/components/Chart/Chart.svelte`).
*   **Логика:** Функция `createMonitor` фильтрует сырые логи (`userMonitorSales`) на клиенте, разбивая их по стадиям и времени.
*   **UX:** Drill-down навигация (клик по графику фильтрует список).

## 2. Профиль пользователя (`views/User.svelte`)

Карточка клиента/резидента.
*   **Редактирование:** Смена статусов, менеджеров.
*   **Тегирование:** Компоненты `TagsSelector` и `TagsSelector2`. Теги имеют иерархию (парсятся по `|` и `>`).
*   **Связи:** История встреч и коммуникаций.

## 3. AI и Мэтчинг (`views/Bundle.svelte` & `views/AI.svelte`)

Интерфейс для работы с ИИ-рекомендациями.
*   Используются запросы `prompt`, `prompt2` для получения связок (intros) между участниками.
