# Менеджер: Запуск и Сборка {id="manager-setup"}

Инструкция по развертыванию приложения `club-manager` для локальной разработки и сборки релизов.

## Предварительные требования

*   **Node.js** 20+
*   **Rust** (последняя стабильная версия) — требуется для сборки Tauri.
*   **XCode** (для macOS) или **C++ Build Tools** (для Windows) — системные зависимости Tauri.

## Настройка окружения

Приложение использует Vite, поэтому переменные окружения должны начинаться с `VITE_`.

1.  Скопируйте `.env.example` в `.env.local` (этот файл игнорируется git).
2.  Настройте URL-адреса API:

```bash
VITE_API_URL=http://localhost:8000       # Основной REST API
VITE_WSO_URL=ws://localhost:8000         # WebSocket сервер
VITE_MEDIA_URL=http://localhost:8005     # Медиа-сервер (аватарки, файлы)
VITE_GRAPHQL_URL=http://localhost:8003   # GraphQL API
```

## Локальный запуск

### Режим браузера (быстрая разработка)

Запускает только фронтенд сервер. Нативные функции Tauri будут недоступны или замоканы.

```bash
npm install
npm run dev
```
Доступно по адресу: `http://localhost:1420`

### Режим Desktop (Tauri)
Запускает приложение в нативном окне. Требует компиляции Rust-части.

```bash
npm run tauri dev
```

## Сборка Production

Сборка создает оптимизированный билд и инсталляторы (dmg, exe, deb).

```bash
npm run tauri build
```

Артефакты сборки появятся в:
*   `src-tauri/target/release/bundle/`

> **Примечание:** Для сборки под конкретную платформу (например, `.exe`) команду нужно выполнять на соответствующей ОС или использовать CI/CD.
